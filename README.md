# Kafeneio Card Game

A full-stack trick-taking card game built with React, TypeScript, Node.js and Socket.IO.

The project features real-time gameplay, AI opponents, bidding mechanics, trump card logic, score tracking and an immersive café-themed game table.

## Features

- Real-time multiplayer architecture
- AI bot opponents
- Bidding / prediction system
- Trump card mechanics
- Round-based scoring
- SVG playing card deck
- Animated game table
- Responsive UI

## Tech Stack

- React
- TypeScript
- Node.js
- Express
- Socket.IO
- Tailwind CSS
- Framer Motion
- Vite
